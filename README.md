# Chaos and Fractals: A Visual Primer

Interactive Python visualizations based on **"Chaos and Fractals: The Mathematics Behind the Computer Graphics"** (AMS, 1989), edited by Robert L. Devaney and Linda Keen.

## Contents

| Chapter | Topic | Author(s) |
|---------|-------|-----------|
| [01](01_dynamics_of_simple_maps/) | Dynamics of Simple Maps | R. L. Devaney |
| [02](02_smale_horseshoe/) | Nonlinear Oscillations and the Smale Horseshoe Map | P. J. Holmes |
| [03](03_fractal_basin_boundaries/) | Fractal Basin Boundaries and Chaotic Attractors | K. T. Alligood & J. A. Yorke |
| [04](04_julia_sets/) | Julia Sets | L. Keen |
| [05](05_mandelbrot_set/) | The Mandelbrot Set | B. Branner |
| [06](06_introduction_to_fractals/) | Introduction to Fractals | J. Harrison |
| [07](07_iterated_function_systems/) | Iterated Function Systems | M. F. Barnsley |

## Setup

```bash
pip install -r requirements.txt
jupyter lab
```

- Python 3.10+
- Jupyter Lab or Jupyter Notebook
